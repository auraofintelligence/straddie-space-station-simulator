# AGENTS.md

This repo is a public, futuristic simulator for an abstract 425m2 Dunwich small-lot terrestrial space-station habitat.

Use Australian English. Keep the exact address out of public copy unless explicitly approved. Public wording may say "Dunwich small-lot test site", "425m2 corner-lot scenario" or "street-edge loading bay".

Treat the project as a simulation-first tool, not a planning claim. Do not imply council approval, building certification, fire-engineering acceptance, insurance approval, sanitation approval or robotics maturity.

The core metaphor is a spacecraft habitat on Earth: fixed modules, known volume, rails, robots, AI concierge, vacuum toilets, separated waste streams, life-support plant, health/wellbeing metrics and agent-readable exports.

Keep the app static and browser-only unless the user asks for a backend. Exports should stay JSON and Markdown first so agents can read and extend layouts.
